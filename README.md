# Sales Data Analysis Project

## Project Overview

This project analyzes a retail sales dataset to identify key trends, patterns, and business insights.
The analysis focuses on data cleaning, SQL-based exploration, and visualization.

---

## Objective

The goal of this project is to:

* Clean and prepare raw sales data
* Analyze the data using SQL queries
* Visualize key trends and patterns
* Extract meaningful business insights

---

## Tools & Technologies

* Python (Pandas)
* SQL (SQLite)
* Matplotlib
* Jupyter Notebook (VS Code)

---

## Process

### 1. Data Cleaning

* Standardized column names
* Converted date columns to datetime format
* Checked for missing values
* Removed duplicate rows

---

### 2. SQL Analysis

Key analyses performed:

* Total sales by category
* Monthly sales trends
* Top 5 products by total sales
* Sales by region and customer segment
* Monthly sales growth

---

### 3. Data Visualization

Created the following visualizations:

* Bar chart: Total sales by category
* Line chart: Monthly sales trend
* Bar chart: Top 5 products by total sales

---

## Key Insights

* The product *Canon imageCLASS 2200 Advanced Copier* is the top contributor to total sales, indicating some degree of product concentration.

* Sales show an overall upward trend over time, suggesting business growth, although there are noticeable fluctuations between months.

* There is clear volatility in monthly sales, with sharp increases and drops, indicating possible seasonality or irregular demand patterns.

* Sales are relatively evenly distributed across categories, although Technology generates the highest total sales.

* Growth is not consistent month-to-month, highlighting potential opportunities for improving sales stability and forecasting.

---

## Conclusion

This project demonstrates how data can be transformed into actionable insights using Python, SQL, and visualization techniques. It highlights the importance of combining data cleaning, structured querying, and visual storytelling in data analysis.
